## Description

  Um sistema simples para organização e automação de estoque/livraria.

## Installation

```bash
$ yarn
```

## Up DB

```
$ yarn dev:up
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
